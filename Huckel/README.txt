qui ho un codice che calcola gli autostati di un'hamiltoninana per un sistema coniugato aperto o chiuso

Si tratta di un toy model, quindi non contiene discriminazione dei fattori beta, ottieni un file di output con gli autovalori che richiedi
Quando viene chiesto, usa 'from' per cominciare un elenco, digita un numero se vuoi un singolo valore oppjure digita 'all' per stampare tutto.
